## Research

I work on the **security testing of large language models** — specifically studying how reinforcement learning can be used to generate adversarial prompts (jailbreaks), and whether causal RL extensions can improve attack effectiveness and efficiency compared to dataset-driven baselines.

**MSc Thesis** *(in progress, Apr 2026)*  
*Security Testing of Large Language Models via Reinforcement Learning and Causal RL*  
Supervisor: Prof. Roberto Pietrantuono · University of Naples Federico II

- Implements and evaluates RL-based jailbreak strategies (REINFORCE, PPO) against LLMs
- Proposes a causal RL extension to improve adversarial attack success and reduce computational cost
- Benchmarks against dataset-based baselines on standard safety evaluation metrics

**Manuscript in preparation** — comparing RL vs. causal RL jailbreak strategies for LLM security evaluation.
